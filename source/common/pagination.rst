Pagination
=================

Requests that return multiple items will be paginated to **10 items** by default.

.. table::
   :width: 100%

   +-------------------+--------------+-------------------------+
   | Field Name        | Type         | Description             |
   +===================+==============+=========================+
   | page              | int          | page number             |
   +-------------------+--------------+-------------------------+
   | limit             | int          | items per page limit    |
   +-------------------+--------------+-------------------------+
